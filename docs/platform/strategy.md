# Платформизация и монетизация

## Принцип платформизации

Всё, что не является core-функциональностью, заменяется на готовые платформы.

### Core Spot (не заменяется)

- Каталог маршрутов.
- Точки и описания.
- Агрегатор пользовательского контента.
- Рекомендации «Поблизости».
- Профили авторов.

### Что можно заменить

| Функциональность | Сейчас в Spot | Чем заменить | Зачем |
| --- | --- | --- | --- |
| Карты и навигация | Своя реализация карты | **Yandex MapKit / 2GIS API** | Карты — это отдельный продукт. Воспроизводить нерационально |
| Хранение медиа (фото точек, обложки) | Object Storage | **VK Cloud Storage / Yandex Object Storage** | S3-совместимое хранилище + CDN из коробки |
| Push-уведомления | Свой сервис | **Firebase Cloud Messaging / VK Push** | Доставка в iOS/Android — не наша зона ответственности |
| SMS с кодом подтверждения | Свой sms-service | **SMSC.ru / SMS Aero** | Стандартная платформенная функция |
| Аналитика поведения | Свой analytics-service | **Яндекс.Метрика / AppMetrica** | Готовая воронка, сегментация, A/B-тесты |
| Email-рассылки (пользовательские дайджесты) | — | **Mindbox / Unisender** | Уменьшение спама, подтверждение открытия |
| Платежи | — | **ЮKassa / CloudPayments** | KYC, эквайринг, чеки |

## Активы Spot для B2B-партнёров

У Spot есть три актива, которые ценны для других компаний.

| Актив | Описание |
| --- | --- |
| **База маршрутов и геоданных** | Тысячи проверенных авторских маршрутов с точками, фотографиями, оценками |
| **Аудитория туристов** | Пользователи, которые планируют поездку и активно ищут, что посмотреть в конкретном городе |
| **Технологическая платформа** | Движок маршрутизации, геопоиск, рейтинговая система |

## Сегменты B2B-клиентов

| Сегмент | Что покупает | Стратегия монетизации |
| --- | --- | --- |
| **Отели и гостиницы** | Доступ к маршрутам для гостей через QR-код в номере, виджет на сайт | Подписка от 5 000 ₽/мес |
| **Туристические агентства** | API для подбора маршрутов клиентам, готовые тур-программы | Pay-per-call: 2 ₽ за вызов API + подписка |
| **Региональные туристические офисы** (Visit Moscow, Visit Sochi) | Приложение «Гид по Сочи» на базе Spot | Большая годовая подписка 1–3 млн ₽ + revenue share |
| **Издатели гайдбуков** (Афиша, Туту) | API маршрутов и точек как контент для своих продуктов | Лицензия на контент: фикс + per-API-call |
| **Авторы и блогеры** (B2C-вторичная) | Премиум-инструменты для создания платных маршрутов, аналитика аудитории | Подписка автора 990 ₽/мес + комиссия 20% с продаж платных туров |

## Стратегии монетизации

Для Spot оптимальна **комбинированная модель**:

| Модель | Применение |
| --- | --- |
| **Подписка (SaaS)** | Основной поток. Отели и турагентства платят ежемесячно за доступ. Предсказуемая выручка, привычная модель для B2B |
| **Pay-per-use** | Для API-доступа и крупных интеграторов. Гибко, привлекает клиентов с нестабильной нагрузкой |
| **Revenue share** | Для региональных туристических офисов с white-label. Spot получает % от выручки региона на туризме |
| **Freemium для авторов** | Базовое создание маршрутов — бесплатно (это даёт поток контента), премиум-инструменты — за деньги |

**Продажа доли** на текущей стадии не выбирается: Spot теряет контроль над развитием продукта. Имеет смысл только при стратегическом партнёрстве (например, с Яндекс.Путешествиями).

## Бизнес-модель: B2B2C

**Основная схема:** Spot → Бизнес (отель, турагентство, регион) → Конечный потребитель (турист).

Spot продаёт доступ к маршрутам отелю → отель показывает маршруты своим гостям через приложение / QR / виджет → турист пользуется маршрутами Spot. Конечный пользователь не платит Spot напрямую, но является ценностью для бизнес-клиента (через его удержание и рекомендации).

Это та же модель, что у **Booking.com** (бизнес — отели, конечный потребитель — путешественник) или **Mindbox** (бизнес покупает рассылку, а конечный пользователь получает письма).

### Дополнительная модель: B2C

Прямые пользователи приложения Spot (туристы и авторы), которые могут переходить на платную подписку. Эта модель работает параллельно и кормит контентную часть платформы.

### Дополнительная модель: B2B (для разработчиков)

API Spot доступен для разработчиков других приложений (тревел-стартапы, агрегаторы).

## Связь между моделями

- Чем больше **авторов** (B2C) → тем больше маршрутов → тем привлекательнее для **отелей и регионов** (B2B2C).
- Чем больше **бизнес-клиентов** → тем больше показов маршрутов туристам → тем выше мотивация авторов создавать контент → растёт **аудитория**.
- Чем больше **аудитории и контента** → тем выше цена API для разработчиков (B2B).

Каждая сторона усиливает другие.
