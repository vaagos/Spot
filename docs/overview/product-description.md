# Описание продукта

## Общая концепция

**Spot** — мобильное приложение с авторскими маршрутами по городам России, которое позволяет пользователю выбрать готовый маршрут и пройти его с помощью встроенной навигации.

Продукт объединяет в одном месте:

- подбор маршрута;
- карту;
- описание локаций;
- последовательность точек;
- навигацию во время прогулки.

## Преимущества по сравнению с альтернативами

| № | Преимущество | Описание |
| --- | --- | --- |
| 1 | Централизация | Вся информация в одном месте |
| 2 | Готовый структурированный маршрут | Не нужно самостоятельно выстраивать последовательность точек |
| 3 | Навигация внутри приложения | Не требуется переходить в сторонние сервисы |
| 4 | Социальный механизм | Рейтинги и подписки формируют доверие |
| 5 | Авторский формат | Маршруты основаны на личном опыте, а не только на популярных достопримечательностях |

## Ключевые функции

### Без них продукт не имеет смысла

- **Каталог маршрутов** (выбор города, список маршрутов, базовая информация: название, описание, длительность). Устраняет необходимость искать маршруты в разных источниках.
- **Страница маршрута** (описание маршрута, последовательность точек, краткое описание каждой точки). Предоставляет структурированную и логически организованную информацию о маршруте.
- **Карта с отображением маршрута** (визуальное отображение точек, порядок прохождения). Позволяет видеть структуру маршрута без ручного планирования.
- **Навигация от точки к точке** (построение маршрута от текущего местоположения, переход от точки к точке). Исключает необходимость переходить в сторонние сервисы.

### Дополнительные

- **Регистрация и авторизация** — персонализация опыта, сохранение данных (избранное, оценки), безопасность доступа.
- **Профиль пользователя и подписки** — формирование доверия к авторам и социальная экосистема.
- **Система оценок маршрутов** — снижает неопределённость при выборе и помогает находить качественный контент.
- **Фильтрация маршрутов** — быстрый подбор под конкретный запрос.
- **Добавление маршрута в «Избранное»** — сохранение для будущего использования.
- **Настройки приложения** — управление уведомлениями, безопасностью, темой.
- **Поддержка** — канал связи с командой приложения.

## Состав MVP

### Реализуется в MVP

- Регистрация и авторизация.
- Каталог маршрутов.
- Страница маршрута.
- Карта с отображением маршрута.
- Навигация от точки к точке.

### Дополнительные функции (по нагрузке команды)

- Добавление собственных маршрутов.
- Профиль пользователя и подписки.
- Система оценок маршрутов.
- Фильтрация маршрутов.
- Добавление в «Избранное».
- Настройки приложения.
- Поддержка.

### Не входит в MVP

- Алгоритмические рекомендации маршрутов.
- Геймификация (награды, достижения, уровни).
- Платная подписка.
- Интеграция музыкальных плейлистов.
- Голосовые комментарии авторов маршрута.

## План развития

Развитие продукта строится по принципу: **Ценность → Вовлечённость → Персонализация → Масштабная монетизация.**

### Версия 1.0 — MVP (запуск и проверка гипотезы)

**Цель:** проверка основной гипотезы — пользователю нужны готовые маршруты с удобной навигацией внутри одного приложения.

**Реализуемый функционал:**

- Каталог маршрутов.
- Страница маршрута.
- Карта с отображением маршрута.
- Навигация от точки к точке.
- (Опционально) регистрация и авторизация.
- (Опционально) добавление своих маршрутов.
- (Опционально) профиль пользователя и подписки.

**Фокус версии:** проверка спроса, анализ поведения пользователей, измерение retention и глубины прохождения маршрутов.

### Версия 1.5 — Повышение вовлечённости

**Цель:** увеличение возвращаемости пользователей и формирование комьюнити.

**Реализуемый функционал:**

- Полноценная регистрация и профиль пользователя.
- Добавление собственных маршрутов с модерацией.
- Подписка на авторов маршрутов.
- Добавление маршрутов в «Избранное».
- Система оценок маршрутов.
- Фильтрация по дополнительным параметрам.

**Фокус версии:** формирование пользовательского контента, рост вовлечённости, увеличение времени в приложении.

### Версия 2.0 — Персонализация и интеллектуальные функции

**Цель:** сделать продукт умнее и увеличить ценность для постоянных пользователей.

**Реализуемый функционал:**

- Алгоритмические рекомендации маршрутов (на основе пройденных маршрутов, рейтингов, интересов).
- Расширенная фильтрация (по настроению, времени суток, погоде).
- Расширенные профили пользователей (статистика, история маршрутов).
- Офлайн-режим с полной загрузкой маршрута.
- Улучшенная аналитика для команды.

**Фокус версии:** повышение персонализации, рост удержания, повышение конкурентоспособности.

### Версия 3.0 — Монетизация и мультимедийный опыт

**Цель:** создание устойчивой бизнес-модели.

**Реализуемый функционал:**

- Платная подписка (эксклюзивные маршруты, специальные подборки).
- Интеграция музыкальных плейлистов (синхронизация с музыкальными сервисами).
- Голосовые комментарии авторов (аудиогиды).
- Элементы геймификации (достижения, уровни, награды).

**Фокус версии:** увеличение LTV пользователя, повышение уникальности продукта, дополнительные источники дохода.
