# Описание проблемы

## Какую проблему решает продукт

Планирование путешествий и прогулок по городам — сложный и трудоёмкий процесс. Пользователи, желающие найти интересный маршрут под свои предпочтения, вынуждены самостоятельно собирать информацию из различных источников.

Особенно остро проблема проявляется:

- при поездке в новый город;
- при ограниченном времени (маршрут на один день);
- при желании получить не шаблонный туристический маршрут, а уникальный опыт.

В результате пользователь тратит значительное время на поиск, анализ и компоновку информации.

## Потребности и боли целевой аудитории

На рынке отсутствует единая платформа, которая:

- объединяет маршруты в структурированном формате;
- предоставляет маршрут сразу в виде готовой последовательности точек на карте;
- содержит описание, личные рекомендации и атмосферу маршрута;
- позволяет быстро выбрать маршрут по интересам, времени и уровню сложности.

Пользователям нужен удобный инструмент, чтобы не тратить время на сбор информации. Есть потребность за пару кликов получить маршрут с картой и описанием — причём маршрут, основанный на реальном опыте других людей.

## Текущие альтернативы

В настоящее время поиск маршрута — довольно разрозненная история. Пользователи используют:

- статьи в журналах (Дзен, Тинькофф Журнал);
- путеводители;
- раздел о городе в Aviasales;
- рекомендации знакомых;
- Reels и TikTok;
- подборки достопримечательностей в Яндекс Картах;
- блоги про путешествия.

## Почему существующие решения недостаточны

| № | Недостаток | Описание |
| --- | --- | --- |
| 1 | Разрозненность информации | Данные находятся в разных источниках, которые не интегрированы между собой |
| 2 | Отсутствие готовой структуры маршрута | Пользователь вынужден самостоятельно выстраивать последовательность точек |
| 3 | Неинтерактивность | Из видео или статей необходимо вручную искать локации на карте |
| 4 | Отсутствие персонализации | Сложно подобрать маршрут под конкретные интересы |
| 5 | Неопределённость по времени и сложности | Часто не указана продолжительность, протяжённость и уровень сложности |
