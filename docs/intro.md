# Spot

**Spot** — мобильное приложение с авторскими маршрутами по городам России. Пользователь выбирает готовый маршрут и проходит его с помощью встроенной навигации. Продукт объединяет в одном месте подбор маршрута, карту, описание локаций, последовательность точек и навигацию во время прогулки.

## Ценность продукта

| Для кого | Что получают |
| --- | --- |
| Пользователь | Возможность быстро выбрать маршрут и следовать ему без самостоятельного планирования и поиска информации в разных источниках |
| Бизнес | Платформенная модель с возможностью монетизации через рекламу партнёров (кафе, музеи, туристические объекты) и платные подписки на эксклюзивные маршруты |

## Ключевые функции (MVP)

- **Регистрация и авторизация** — по номеру телефона с подтверждением через SMS
- **Каталог маршрутов** — три раздела (Популярное, Новое, Поблизости) с карточками маршрутов
- **Страница маршрута** — описание, последовательность точек, фото, рекомендации
- **Карта с маршрутом** — визуальное отображение точек и порядка прохождения
- **Навигация от точки к точке** — построение маршрута от текущего местоположения

## Технический стек

| Компонент | Технология |
| --- | --- |
| Платформы | iOS ≥ 15, Android ≥ 8 |
| Backend | Java/Kotlin или Python |
| Основное хранилище | PostgreSQL |
| Кэш / сессии | Redis |
| Очередь событий | Apache Kafka |
| Карты и навигация | Yandex Maps API |
| Хранение медиа | S3-совместимое (VK Cloud / Yandex Object Storage) |
| Push-уведомления | Firebase Cloud Messaging / VK Push |
| SMS | SMSC.ru / SMS Aero |
| Аналитика | AppMetrica |

## План развития

| Версия | Цель | Сроки |
| --- | --- | --- |
| 1.0 — MVP | Проверка гипотезы спроса на готовые маршруты | 0–6 месяцев |
| 1.5 — Вовлечённость | UGC-маршруты, оценки, подписки, избранное | 6–12 месяцев |
| 2.0 — Персонализация | Алгоритмические рекомендации, офлайн-режим | 12–18 месяцев |
| 3.0 — Монетизация | Платная подписка, аудиогиды, геймификация | После 18 месяцев |

## Бизнес-цели (первый год)

- 30 000 зарегистрированных пользователей в 5 крупных городах России
- 1 000 пользовательских маршрутов
- MAU/Registered Users ≥ 40%
- 50 партнёрских соглашений и 5% конверсия в платную подписку
